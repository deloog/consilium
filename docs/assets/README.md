# Consilium Assets

This directory contains project assets.

## Logo

- `consilium-logo.svg` - Main project logo (SVG format)
- Recommended size: 200x200px for README
- Colors:
  - Primary: #e94560 (Coral Red)
  - Secondary: #00d9ff (Cyan)
  - Background: #1a1a2e (Dark Navy)
  - Accent: #0f3460 (Deep Blue)

## Design Concept

The logo represents:
- **Shield** - Security and protection
- **Check Mark** - Approval and quality assurance
- **6 Dots** - The six-phase deliberation process (6 agents collaborating)
- **Circle** - Unity and completeness

## Usage

Feel free to use the logo for:
- Project documentation
- Promotional materials
- Presentation slides
- Social media posts

Please maintain the original colors and proportions when using the logo.
